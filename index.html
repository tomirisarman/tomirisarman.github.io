Способности

Уверенно разрабатываю сайты - как front-end (наружность сайта), так и back-end (логика сайта).
Для front-end нужны HTML+CSS+JS. Так же для адаптации под мобильные устройства использую Flexbox, Grid CSS, Bootstrap framework. Адаптивность этого сайта реализована через Grid - очень удобно и не затрачивается много ресурсов.
Back-end пишу на PHP и работаю с реляционными базами данных MySQL. Времена меняются вместе с трендами в разработке, поэтому на данный момент Python Django у меня в процессе изучения.
Умею пользоваться Photoshop, в основном для обыденных целей. Иногда редактирую видео в Davinci Resolve.
Начинаю заниматься трехмерным моделированием в Autodesk 3Ds Max - строю себе виллу на берегу моря. Хоть где-то она у меня будет.
